# RO-UPLOADER

RO-UPLOADER is a tool developed by 5nz to help you mass upload assets to ROBLOX without any issues. This uploader uses the Open Cloud API Key.

## Features

- **Decal Mass Uploader:** Easily upload multiple decals in one go.
